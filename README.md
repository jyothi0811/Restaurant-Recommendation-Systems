# Restaurant Recommendation System

This project is a Machine Learning-based Restaurant Recommendation System that predicts and suggests restaurants to customers based on their preferences, location, and past order behavior.

## Objective
To build an intelligent system that recommends restaurants using data-driven techniques and machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing and cleaning
- Feature engineering (order count, total spending, average spending)
- Machine learning model for prediction
- Recommendation generation based on probability scores
- Top-N restaurant recommendations

## How It Works
1. Collect customer, location, and order data  
2. Perform data preprocessing  
3. Create meaningful features  
4. Train machine learning model  
5. Predict customer preferences  
6. Recommend top restaurants based on scores  

## Outcome
- Successfully generated restaurant recommendations  
- Improved understanding of machine learning and recommendation systems  

## Learning Outcomes
- Hands-on experience with machine learning models  
- Improved data analysis and feature engineering skills  
- Understanding of recommendation system logic
